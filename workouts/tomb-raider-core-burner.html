Tomb Raider Core Burner

Inspired by: Lara Croft

Functional strength, core power, and agility like an adventurer on the move

Difficulty: Medium

  • Warm-Up: Jump rope 2 mins + dynamic stretches
  • Hanging Leg Raises: 3 x 10
  • Push-Ups with Shoulder Taps: 3 x 12
  • Bulgarian Split Squats: 3 x 8 each leg
  • V-Ups: 3 x 15
  • Side Planks with Hip Lifts: 3 x 10 each side
  • Jump Lunges: 3 x 12
  • Cool Down: Stretch glutes, hip flexors, abs
